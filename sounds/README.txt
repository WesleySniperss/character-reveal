Drop audio files here to play during the reveal effect.

Supported formats: mp3, ogg, wav, flac, webm, m4a, aac

PER-STYLE SOUNDS
Name the file after the style ID to play it only for that style:

  minimal.mp3
  tarantino.ogg
  wantedpost.wav
  borderlands.mp3
  heraldry.ogg
  darksouls.mp3
  manuscript.ogg
  spotlight.wav

FALLBACK
Any file with a name that doesn't match a style ID will be used
as a fallback for all styles that have no dedicated sound file.
